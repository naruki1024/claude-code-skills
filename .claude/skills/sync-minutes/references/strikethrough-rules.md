# 取り消し線の適用ルール

## 基本原則

**既存の記載は絶対に削除しない**。変更履歴を残すため、必ず取り消し線を使用する。

## マークダウン形式

```markdown
~~取り消したいテキスト~~
```

## 適用パターン

### 1. 単純な文の修正

```markdown
# Before
視聴期間は90日間。

# After
~~視聴期間は90日間。~~

視聴期間は120日間に変更。

<!-- 変更: 20260106_定例.md -->
```

### 2. 箇条書きの項目修正

```markdown
# Before
* **価格**: ¥9,800

# After
* ~~**価格**: ¥9,800~~
* **価格**: ¥9,980

<!-- 変更: 20260106_定例.md -->
```

### 3. 段落全体の修正

```markdown
# Before
本機能では、ユーザーが自由にプランを変更できる。
変更時の日割り計算は行わない。

# After
~~本機能では、ユーザーが自由にプランを変更できる。~~
~~変更時の日割り計算は行わない。~~

プラン変更は月末のみ可能。変更は翌月から適用される。

<!-- 変更: 20260106_定例.md -->
```

### 4. 表の中の修正

```markdown
# Before
| プラン | 価格 | 視聴期間 |
|--------|------|---------|
| スタンダード | ¥19,800 | 90日 |

# After
| プラン | 価格 | 視聴期間 |
|--------|------|---------|
| スタンダード | ¥19,800 | ~~90日~~ 120日 |

<!-- 変更: 20260106_定例.md -->
```

### 5. MVPスコープの変更

```markdown
# MVP対象から除外
~~**MVP対象**~~

**MVP対象外**（次フェーズで実装）

<!-- スコープ変更: 20260106_定例.md -->
```

```markdown
# MVP対象に追加
~~①初期見送り~~

**MVP対象**（簡易実装）

<!-- スコープ変更: 20260106_定例.md -->
```

## 複数回の変更がある場合

履歴が分かるよう、変更の積み重ねを残す:

```markdown
### 視聴期間

~~60日~~

<!-- 変更: 20251201_定例.md -->

~~90日~~

<!-- 変更: 20251215_定例.md -->

120日（現行）

<!-- 変更: 20260106_定例.md -->
```

## 注意事項

- 取り消し線の中にマークダウン記法（**太字**など）がある場合も、そのまま `~~` で囲む
- 見出し（`#`、`##`など）自体は取り消し線にしない。見出しの内容のみを修正
- コードブロック内のテキストは通常のテキストとして扱う
