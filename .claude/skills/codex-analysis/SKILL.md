# Codex コードベース分析

## 概要

このSkillはCodex CLIをサブエージェントとして活用し、複雑なコード分析タスクを委譲します。

## 実行可能な分析

1. **バグ調査**: エラーの根本原因を特定
2. **セキュリティレビュー**: 脆弱性を検出
3. **パフォーマンス最適化**: 遅いコード部分を特定
4. **アーキテクチャレビュー**: 設計パターンの評価
5. **コードベース理解**: プロジェクト構造の把握

## 実行方法

```bash
codex exec --full-auto --sandbox read-only --cd /home/satoshi/workspace/heygen-interactive "<分析リクエスト>"
```

## 使用例

- 「Codexでこのエラーの原因を調査して」
- 「RAGパイプラインのパフォーマンスをCodexで分析して」
- 「セキュリティの問題がないかCodexでレビューして」
