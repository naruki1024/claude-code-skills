# コード説明コマンド

## タスク
指定されたコード（$ARGUMENTS）について詳しく説明します。

## 説明内容

### 1. 概要
- このコードの目的
- 全体的なアーキテクチャ

### 2. 主要なコンポーネント
- 関数/クラスの役割
- データフロー
- 依存関係

### 3. 重要な処理
- キーとなるアルゴリズム
- 注意が必要な部分
- エラーハンドリング

### 4. 使用例
- 基本的な使い方
- オプションパラメータ
- 戻り値の説明

## 出力形式
- 初心者にも分かりやすい説明
- 必要に応じてコード例を含める
- 関連するドキュメントへのリンク（あれば）
