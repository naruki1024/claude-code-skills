# ドキュメント取得コマンド

## タスク
指定されたURLからHTMLコンテンツを取得し、マークダウン形式に変換して保存します。

## 手順

1. **引数を確認**
   - URL: $1
   - 出力ファイル名: $2（省略時はURLから自動生成）

2. **HTMLを取得**
   ```bash
   curl -s "$1" -o /tmp/fetched_page.html
   ```

3. **HTMLからテキストを抽出**
   - HTMLタグを除去
   - 見出し、段落、リストなどの構造を保持
   - コードブロックを適切にフォーマット

4. **マークダウンファイルとして保存**
   - 指定されたファイル名で保存
   - ファイル名が未指定の場合、URLのパスから生成

## 注意事項
- 大きなページは複数のリクエストに分割される場合があります
- JavaScriptでレンダリングされるコンテンツは取得できません
