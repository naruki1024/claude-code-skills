# Claude Code Legal and Compliance Information

Based on the documentation, here is the legal and compliance information for Claude Code:

## Legal Contracts

### Licenses
Claude Code usage is subject to:
- **Commercial Terms** - For Team, Enterprise, and Claude API users
  - Link: https://www.anthropic.com/legal/commercial-terms
- **Consumer Terms** - For Free, Pro, and Max users
  - Link: https://www.anthropic.com/legal/consumer-terms

### Commercial Agreements
Existing commercial contracts apply to Claude Code usage whether you:
- Use Claude API directly (1st party), or
- Access through AWS Bedrock or Google Vertex (3rd party)

Unless separate agreements are in place, your current commercial contract governs Claude Code usage.

## Compliance

### Healthcare Compliance (BAA)
If your organization has a **Business Associate Agreement (BAA)** with Anthropic:
- The BAA automatically extends to cover Claude Code
- Requirement: BAA must be executed and **Zero Data Retention (ZDR)** must be enabled
- Coverage applies to API traffic flowing through Claude Code
